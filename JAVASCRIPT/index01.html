项目一：对话框和页面输出 —— 初步体验

有两张网页，每张网页上都有一个"问候"按钮。在第一张网页上点击该按钮，弹出对话框"张三向您问好！"，点击对话框上的"确定"按钮后，紧接着在该网页上打印"张三欢迎您光临本站！"。在第二张网页上做同样的操作，显示结果相同，只不过信息中把"张三"显示为了"李四"。项目要求代码的复用性和可维护性良好。

项目情境

  • 熟悉JavaScript代码内嵌在HTML代码中的书写方法。
  • 熟悉JavaScript代码的执行时机。
  • 初步了解事件触发的概念。
  • 初步了解函数和函数调用的方法。
  • 熟悉JavaScript文件的编写和引入的方法。
  • 能够初步理解JavaScript的运行原理。
  • 能够初步建立模块化编程的思维。
  • 培养协同合作开发项目的能力。

学习目标

  • 打开页面，立即弹出一个问候对话框。
  • 掌握JavaScript在HTML中的基本嵌入方法。
  • 了解事件触发和函数调用的基本概念。
  • 掌握JavaScript文件的编写和引入方法。
  • 理解代码模块化的重要性。

任务1：弹出对话框

相关知识

  • 在HTML中，任何标签都可以看成是一个对象，例如body就是一个对象。这些对象一般都有属性、事件和方法。
  • 如果要想在页面被引导后执行一个任务，可以给<body>标签添加一个onload事件，此事件在body对象被加载完成后被触发，写法形如：onload="要执行的代码";
  • alert("参数")功能是弹出一个对话框，对话框中的内容就是该函数中设置的参数。
  • JavaScript对大小写是敏感的，所以alert()必须全部小写。

任务实现

在<body>标签内部输入以下代码：

HTML代码示例
<body onload="alert('你好！')">

任务2：在页面上打印欢迎词

相关知识

  • document表示的是文档对象，每个载入浏览器的 HTML 文档都会成为 Document 对象。
  • document对象有很多方法，write方法是其中之一，表示在文档中打印信息内容。

任务实现

在<body>标签内部输入以下代码：

HTML代码示例
<body onload="document.write('欢迎光临本站！')">

任务3：先弹对话框再打印

相关知识

  • 在JavaScript中，如果功能块有多个语句，这些语句中间用分号隔开。实际上，每一个完整的功能语句结束处都应该写上分号，哪怕只有一条语句，也应该写上。

任务实现

在<body>标签内部输入以下代码：

HTML代码示例
<body onload="alert('你好！'); document.write('欢迎光临本站！');">

任务4：点击按钮执行任务3

相关知识

  • document.write()执行时，将重写当前页面。所以，页面上原有的内容将全部消失，只留下document.write()写下的内容。在本任务中，页面中最后看不到原来的"问候"按钮就是这个原因。

任务实现

在<body>标签对中编写按钮代码，并在按钮代码中添加onclick事件属性，并编写执行任务代码：

HTML代码示例
<input type="button" name="button" id="button" value="问候" onclick="alert('你好！'); document.write('欢迎光临本站！');" />

任务5：任务模块化

相关知识

  • 代码模块化：将代码"打包"为函数的形式
  • Javascript与HTML混编：<script></script>标签的使用
  • 注释：//…… 注释单行；/*……*/ 注释多行

任务实现

1. 在<body>标签对中编写按钮代码：

HTML代码示例
<input type="button" name="button" id="button" value="问候" onclick="hello()" />

2. 在页面的<head></head>标签内部编写代码：

JavaScript代码示例
<script type="text/javascript">
// 下面定义了一个函数hello
function hello(){
    alert('你好！'); 
    document.write('欢迎光临本站!');
}
</script>

任务6：调用灵活化

相关知识

在任务5中，我们将代码"打包"为一个函数，实现了模块化，但是还不够灵活。例如要改变显示的内容，就必须要修改函数体中的代码。

程序设计中常常通过在编写函数模块时定义参数（一般是用逗号隔开的变量列表），而在调用函数时传入具体的数值的方法实现函数功能的灵活性和可扩展性。

任务实现

1. 在<body>标签对中编写按钮代码：

HTML代码示例
<input type="button" name="button" id="button" value="问候" onclick="hello('你好','欢迎光临本站！');" />

2. 在页面的<head></head>标签内部编写代码：

JavaScript代码示例
<script type="text/javascript">
function hello(str1, str2){
    alert(str1);
    document.write(str2);
}
</script>

任务7："项目一"的实现

相关知识

  • js文件：扩展名为js的文件是用JavaScript编写的客户端脚本文件，它不是一般的网页文件，一般不能直接运行打开，而是配合网页来使用。它常常用来实现某些功能，这些功能代码可以被多个不同的网页调用。在js文件中编写JavaScript脚本时，不要用<script>标签对包括。
  • 引用js文件：在页面中，只有引入js文件才能使用该文件中的JavaScript代码。方法是在<script>标签中添加src属性，其值就是该js文件的路径。

项目分析

很显然，在完成本项目的任务6后，要想完成本项目就非常简单了，只要制作两个页面，并分别在这两个页面中用本项目任务6中的方法就可以实现。但是，在最终实现本项目时，希望项目中的代码的复用性和可维护性更强。试想，既然这两个页面调用的函数都是一样的，那么可不可以把这个函数独立出来共享呢？答案是可以。

项目实现步骤

  1. 建立一个文件hello.js（注意扩展名为js）。
  2. 在hello.js中编写如下函数代码并保存文件。
  3. 建立两个页面，分别为hello1.html和hello2.html（为简单起见，将这三个文件放置在同一个目录下）。
  4. 在hello1.html中，在<body>标签对内部编写按钮代码。
  5. 在页面的<head></head>标签内部编写引入js文件的代码。
  6. 在hello2.html中做同样的操作，只是将"张三"改为"李四"。
hello.js文件内容
// hello.js文件
function hello(str1, str2){
    alert(str1);
    document.write(str2);
}
hello1.html文件内容
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>hello1</title>
    <script type="text/javascript" src="hello.js"></script>
</head>
<body>
    <input type="button" name="button" id="button" value="问候" 
           onclick="hello('张三向您问好','张三欢迎您光临本站！');" />
</body>
</html>
hello2.html文件内容
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>hello2</title>
    <script type="text/javascript" src="hello.js"></script>
</head>
<body>
    <input type="button" name="button" id="button" value="问候" 
           onclick="hello('李四向您问好','李四欢迎您光临本站！');" />
</body>
</html>

项目总结

通过本项目，我们学习了：

  • JavaScript在HTML中的嵌入方法
  • alert()对话框和document.write()页面输出
  • 事件触发（onload、onclick）
  • 函数定义和调用
  • 函数参数的使用
  • 外部JavaScript文件的创建和引用
  • 代码模块化和复用性设计

这个项目展示了良好的代码复用性和可维护性，相同的hello函数可以被多个页面使用，只需传入不同的参数即可实现不同的功能。
