项目二：简单计算器 —— 判断结构

设计一个简单的计算器，能进行加、减、乘、除四则运算。当用户运行网页时，依次弹出三个输入对话框，分别要求输入两个数和一个运算符号（+、-、×、÷）。输入无误时，在页面上打印计算算式。在进行除法运算时，除数不能为0，否则打印错误信息。如果输入数字格式错误或者输入的运算符号错误都提示相应的不能运算信息。

项目情境

  • 熟悉变量和常量的概念与使用方法。
  • 熟悉基本数据类型的概念。
  • 熟悉数据类型的转换。
  • 熟悉关系运算符、逻辑运算符、算术运算符的使用方法。
  • 熟悉判断结构的使用方法。
  • 熟悉几个内置函数的使用方法。
  • 培养逻辑思维和问题解决能力。
  • 培养耐心和细致的工作态度。

学习目标

  • 理解变量和常量的概念，掌握其定义和使用方法。
  • 掌握JavaScript的基本数据类型。
  • 理解并掌握数据类型的转换。
  • 掌握各种运算符的使用方法和优先级。
  • 掌握判断结构（if语句、switch语句）的使用方法。
  • 能够使用prompt()函数获取用户输入。
  • 能够处理用户输入的验证和错误处理。
  • 能够实现一个简单的四则运算计算器。

任务1：认识变量和字符串

打开页面，打印程序中指定的姓名和性别。

相关知识

1. 变量的定义

定义一个变量用关键字var，例如：

var x;
var y;

一个var也可以同时定义多个变量，这些变量间用逗号隔开，例如：

var x, y, z;

2. 变量的赋值

变量的赋值就是把值存储在变量所在的内存单元中，方法是用"="运算符。赋值可以是在定义变量时，例如： 

var x = 0;
var y = 1, z = 2;

也可以是在变量定义后使用时才赋值，例如：

var x;
x = 0;

3. 变量的命名

变量的名称必须是由ASCII字符或者下划线（_）开头，第一个字符不能是数字，但其后可以是数字或者其他字母。例如如下变量的命名是合法的： 

a
A
_a
a_1

注意：JavaScript语言对大小写是敏感的，所以变量a和变量A是两个不同的变量。变量名不能定义为JavaScript的保留字。例如不能定义一个变量的名称为var或者function。

4. 常量

与变量对应的是常量。常量用来表示一个固定不变的值。比如下面这些这些都是常量：

123
25.78
"张三"
true

在ECMAScript 6以上版本中，常量可以使用关键字const来定义，如const PI = 3.14。常量在声明时可以使用大小写，但通常情况下全部用大写字母。常量一旦赋值，就不允许再被改变。

5. 字符串型

  • 字符串的定义：字符串必须用双引号或者单引号包括起来。
  • 转义符：转义符"\"用于表示特殊字符，如"\n"表示换行。
  • 连接运算符：连接运算符"+"用于连接字符串。

任务2：输入对话框

打开页面，弹出一个输入对话框，要求输入姓名。输入确定后再次弹出输入对话框，要求输入性别。输入确定后，在页面打印刚刚输入的信息。

相关知识

1. Window对象

window对象表示浏览器窗口。任何一个全局函数或变量都是window对象的属性，所以使用时，该对象常常省略不写。

2. prompt方法

prompt方法的功能是弹出一个输入对话框，格式是：

prompt(<提示信息> [, 默认值])

其返回的值就是用户在对话框中输入的值，返回值的类型是字符串型。

任务3：判断是否为整数

打开页面，弹出一个输入对话框，要求输入一个整数。如果输入的不是一个整数，则给出错误提示。

相关知识

1. 表达式

表达式可以是常量或者变量，也可以是由常量、变量和运算符号组成的语句。赋值语句也是表达式。比如下面这些都是合法的表达式： 

4.56
true
a > b
a + b
a = 10

2. 布尔数据类型

布尔（boolean）是一种基本数据类型，表示真或者假，通俗的理解为是或者非，对或者错等。它常常用于判断一个结果的是非性。它的值只有两个：true（真）和false（假）。例如：8 > 4其结果就是false。再如：a < 6如果a是一个比6小的值，则结果就是true，如果a是一个比6大或者等于6，结果就是false。

3. 关系运算符

运算符 描述 示例 结果
== 等于 5 == 5 true
=== 严格等于（值和类型） 5 === "5" false
!= 不等于 5 != 4 true
!== 严格不等于 5 !== "5" true
> 大于 5 > 4 true
< 小于 5 < 4 false
>= 大于等于 5 >= 5 true
<= 小于等于 5 <= 4 false

4. 判断语句

4.1 if语句

（1）if语句

if(<条件>) {
    [语句块]
}

（2）if...else语句

if(<条件>) {
    [语句块1]
} else {
    [语句块2]
}

（3）if...else if...else语句

if(<条件1>) {
    [语句块1]
} else if(<条件2>) {
    [语句块2]
} else if(<条件n>) {
    [语句块n]
} else {
    [语句块n+1]
}

（4）if语句的嵌套

if语句允许嵌套，嵌套时注意if与else的匹配，不要出现交叉嵌套的现象。

4.2 switch语句
switch(<变量或表达式>) {
    case <数值1>:
        [语句块1]
        break;
    case <数值2>:
        [语句块2]
        break;
    ...
    case <数值n>:
        [语句块n]
        break;
    default:
        [语句块n+1]
}

5. 三目运算

三目运算是根据条件执行两个语句中的其中一个，格式如下：

布尔表达式 ? 语句1 : 语句2

当"布尔表达式"值为真时，执行"语句1"；当"布尔表达式"值为假时，执行"语句2"。

任务实现

在<head>标签内部输入以下代码：

JavaScript代码示例
<script type="text/javascript">
var num = window.prompt("请输入一个整数：", "");
if(parseInt(num) == num) {
    document.write(num + "是整数");
} else {
    document.write(num + "不是整数");
}
</script>

任务4：判断指定范围的奇偶数

当页面被打开时，弹出一个输入对话框，要求输入一个指定范围的整数（10~100之间）。如果输入的内容为奇数且在指定范围内，则单击"确定"按钮后打印信息，提示该数为奇数。如果输入的内容为偶数且在指定范围内，则单击"确定"按钮后打印信息，提示该数为偶数。如果输入的内容为以下情况中的一种，就视为不合法输入，单击"确定"按钮后打印信息，提示错误。

  1. 输入的内容不是数字。
  2. 输入的内容是数字，但不是整数。
  3. 未进行任何输入。
  4. 不在指定范围。

相关知识

1. 逻辑运算符

运算符 描述 示例 结果
&& 逻辑与 (5 > 3) && (5 > 2) true
|| 逻辑或 (5 > 3) || (5 < 2) true
! 逻辑非 !(5 > 3) false

2. 算术运算符

运算符 描述 示例 结果
+ 加法 5 + 3 8
- 减法 5 - 3 2
* 乘法 5 * 3 15
/ 除法 6 / 3 2
% 取模（求余） 5 % 3 2
++ 自增 x = 5; x++ x = 6
-- 自减 x = 5; x-- x = 4

3. 运算符的优先级

从高到低：括号 → 自增/自减 → 乘/除/取模 → 加/减 → 关系运算符 → 相等运算符 → 逻辑与 → 逻辑或 → 赋值运算符。

4. isNaN函数

isNaN()函数接收一个参数，用于检查这个参数是否"不是数值"。isNaN()函数在接收到一个值之后，会尝试将这个值转换为数值，如果能成功转换，则返回false，否则返回true。该函数的基本语法是isNaN(value)，其中value就是要被检查的值。

需要注意的是，当value为空格（或者由空格组成的字符串）、null值时，该函数返回的值也是false，原因是isNaN()函数将这些值转换为了数字0。

任务实现

在<head>标签输入下列代码：

JavaScript代码示例
<script>
var num = window.prompt("请输入一个整数(10~100)：", "");
if(parseInt(num) == num && num >= 10 && num <= 100) {
    var res = num % 2;
    if(res == 0) {
        document.write(num + "是偶数");
    } else {
        document.write(num + "是奇数");
    }
} else {
    document.write("输入的数字不符合要求！");
}
</script>

任务5：加法运算器

打开页面，弹出输入对话框，要求输入一个被加数。输入确定后，再次弹出输入对话框，要求输入一个加数。输入确定后，打印这两个数的和。如果这两个数中只要有一个输入格式不正确，或者未输入任何值，都给出错误信息。

相关知识

1. 数字型

JavaScript中的数字可以是整数或浮点数，例如：123、3.14、-5等。

2. 数据类型的转换

2.1 隐式类型转换：JavaScript在某些情况下会自动进行类型转换，例如在字符串和数字之间使用"+"运算符时，数字会被转换为字符串。

2.2 显示类型转换：使用parseInt()、parseFloat()、Number()等函数进行显式转换。

3. 查看数据类型

可以使用typeof运算符查看一个数据的数据类型。typeof运算符有以下两种使用方式： 

typeof(带运算符号的表达式或者变量)
typeof 变量名

任务实现

在<head>标签对中输入如下代码：

JavaScript代码示例
<script>
var num1 = window.prompt("请输入被加数：", 0);
var num2 = window.prompt("请输入加数：", 0);
if(parseFloat(num1) == num1 && parseFloat(num2) == num2) {
    var res = parseFloat(num1) + parseFloat(num2);
    document.write(num1 + "+" + num2 + "=" + res);
} else {
    document.write("输入的数字格式不正确！");
}
</script>

任务6："项目2"的实现

在本任务中，需要提供输入对话框，以及检验用户输入的内容是否合法。只有当用户输入的所有内容都合法时才进行计算。当输入的数字合法但运算符号不合法时就要进行处理。当进行算术除法运算时，还要考虑除数为0的情况。因为要进行算术运算，并且考虑到小数的问题，所以还要把输入的数字转换为浮点数。最后，为了更完整地显示结果，还需要把数字类型和字符串类型的数据连接在一起输出。

项目分析

本项目需要实现一个完整的四则运算计算器，包括：

  1. 获取用户输入的两个数字和一个运算符
  2. 验证输入的数字是否合法（是否为数字）
  3. 验证运算符是否合法（是否为+、-、×、÷）
  4. 对于除法运算，检查除数是否为0
  5. 根据运算符执行相应的计算
  6. 显示计算结果或错误信息

项目实现

在<head>标签对中输入以下代码：

JavaScript代码示例
<script>
var num1 = window.prompt("请输入第一个数：", 0);
var num2 = window.prompt("请输入第二个数：", 0);
var op = window.prompt("请输入一个运算符号(+、-、×、÷)：", "+");
if(parseFloat(num1) == num1 && parseFloat(num2) == num2) {
    num1 = parseFloat(num1);
    num2 = parseFloat(num2);
    var flag = 0;
    var res = "";
    switch(op) {
        case "+":
            res = num1 + num2;
            break;
        case "-":
            res = num1 - num2;
            break;
        case "×":
            res = num1 * num2;
            break;
        case "÷":
            if(num2 != 0) { // 如果除数不为0
                res = num1 / num2;
            } else {
                flag = 1;
                res = "除数不能为0！";
            }
            break;
        default: // 如果输入的运算符号不是+、-、×、÷中的任何一个
            flag = 1;
            res = "输入的运算符号错误！不能计算！";
    }
    if(flag == 0) {
        document.write(num1 + op + num2 + "=" + res);
    } else {
        document.write(res);
    }
} else {
    document.write("输入的数字格式错误！不能计算！");
}
</script>
