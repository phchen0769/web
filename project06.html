表格与表单知识手册

表格和表单是网页中用于数据展示和数据收集的重要元素，在"千年之恋"注册页面制作中起着关键作用。

表格与表单的作用

  • 数据展示 - 表格用于清晰地展示结构化数据
  • 数据收集 - 表单用于收集用户输入的信息
  • 用户交互 - 表单提供用户与网站交互的界面
  • 数据验证 - 表单可以验证用户输入的数据格式

表格的创建

表格用于在网页中展示结构化数据，由行和列组成。

表格的基本结构

创建table表格，离不开以下3对标签：

  • <table></table> - 定义一个表格
  • <tr></tr> - 定义表格中的一行，嵌套在<table></table>中
  • <td></td> - 定义表格中的单元格，嵌套在<tr></tr>中
<table> <tr> <td>单元格1</td> <td>单元格2</td> </tr> <tr> <td>单元格3</td> <td>单元格4</td> </tr> </table>

tr标签属性

tr标签用于定义表格中的行，可以设置行的样式。

常用tr标签属性

属性 说明 示例
align 设置行内容的水平对齐方式 align="center"
valign 设置行内容的垂直对齐方式 valign="middle"
bgcolor 设置行的背景颜色 bgcolor="#f0f0f0"

注意：

  1. <tr>标签无宽度属性width，其宽度取决于表格标签<table>。
  2. 可以对<tr>标签应用valign属性，来设置一行内容的垂直对齐方式。
  3. 不建议对<tr>标签应用background属性，以免出现兼容性问题。

td标签属性

td标签用于定义表格中的单元格，可以设置单元格的样式。

常用td标签属性

属性 说明 示例
width 设置单元格的宽度 width="100"
height 设置单元格的高度 height="50"
align 设置单元格内容的水平对齐方式 align="center"
valign 设置单元格内容的垂直对齐方式 valign="middle"
colspan 设置单元格横跨的列数 colspan="2"
rowspan 设置单元格横跨的行数 rowspan="2"
bgcolor 设置单元格的背景颜色 bgcolor="#f0f0f0"

注意：当对某一个<td>标签应用width属性设置宽度时，该列中的所有单元格均会以设置的宽度显示。当对某一个<td>标签应用height属性设置高度时，该行中的所有单元格均会以设置的高度显示。

th标签

th标签用于定义表格的表头单元格。

th标签的特点

  • th标签用来设置表头
  • 其文本默认加粗居中显示
  • 用法与td标签类似，可以应用相同的属性
  • 有助于提高表格的可访问性和语义化
HTML 代码:
<table> 
  <tr> 
    <th>姓名</th> 
    <th>年龄</th> 
    <th>城市</th> 
  </tr> 
  <tr> 
    <td>张三</td> 
    <td>25</td> 
    <td>北京</td> 
  </tr> 
</table></pre>

CSS控制表格边框

使用CSS可以更灵活地控制表格的边框样式。

CSS边框属性

  • border-width - 设置边框宽度
  • border-style - 设置边框样式
  • border-color - 设置边框颜色
  • border - 简写属性，同时设置宽度、样式和颜色
/* 设置表格边框 */ table { width: 280px; height: 280px; border: 1px solid #F00; } /* 设置单元格边框 */ td, th { border: 1px solid #F00; }

注意：在设置表格的边框时，也要给单元格设置相应的边框。

CSS控制单元格边距

使用CSS可以控制单元格内容与边框之间的距离。

CSS内边距属性

CSS 代码:
/* 设置单元格内边距 */
td { 
  padding: 20px; 
}</pre>

注意：

  1. 行标签<tr>无内边距属性padding和外边距属性margin。
  2. 外边距属性margin对单元格无效，要想设置相邻单元格边框之间的距离，只能对<table>标签应用HTML标签属性cellspacing。

新增input控件属性

HTML5新增了多种input控件属性，可以增强表单的功能和用户体验。

HTML5新增的input属性

属性 说明 示例
autofocus 页面加载时自动获得焦点 <input type="text" autofocus>
placeholder 输入框内的提示文本 <input type="text" placeholder="请输入姓名">
required 规定输入字段不能为空 <input type="text" required>
pattern 规定输入值的模式（正则表达式） <input type="text" pattern="[A-Za-z]{3}">
min, max, step 规定输入值的最小值、最大值和步长 <input type="number" min="1" max="10" step="1">
multiple 允许输入多个值 <input type="file" multiple>
list 引用包含输入选项的datalist <input type="text" list="browsers">

CSS控制表单样式

使用CSS可以美化表单，提升用户体验。

常用的表单样式属性

  • font-family, font-size, color - 控制字体样式
  • border, border-radius - 控制边框样式
  • background, background-color - 控制背景样式
  • padding, margin - 控制内边距和外边距
  • width, height - 控制尺寸
  • :focus - 控制获得焦点时的样式
/* 美化输入框 */ input[type="text"], input[type="password"] { width: 100%; padding: 10px; border: 1px solid #ddd; border-radius: 4px; font-size: 16px; } /* 输入框获得焦点时的样式 */ input[type="text"]:focus, input[type="password"]:focus { border-color: #3498db; box-shadow: 0 0 5px rgba(52, 152, 219, 0.5); outline: none; } /* 美化按钮 */ input[type="submit"], input[type="reset"] { background-color: #3498db; color: white; border: none; padding: 10px 20px; border-radius: 4px; cursor: pointer; } /* 按钮悬停效果 */ input[type="submit"]:hover, input[type="reset"]:hover { background-color: #2980b9; }
