CSS知识点完整参考手册

CSS（层叠样式表）是一种用于描述HTML文档呈现方式的样式表语言。它控制网页的布局、颜色、字体等视觉表现。

CSS的作用

  • 内容与表现分离 - HTML负责结构，CSS负责样式
  • 提高开发效率 - 一套样式可用于多个页面
  • 增强可维护性 - 修改样式无需改动HTML结构
  • 实现响应式设计 - 使网页适应不同设备屏幕

CSS的版本

  • CSS1 - 1996年发布，包含基础样式属性
  • CSS2 - 1998年发布，增加定位、媒体类型等
  • CSS2.1 - 2011年发布，修正CSS2的问题
  • CSS3 - 模块化发布，包含众多新特性

结构与表现分离

结构与表现分离是Web标准的核心思想之一，指的是将HTML文档的结构与CSS样式分开处理。

为什么要将结构和表现分离？

  • 提高代码可维护性 - 修改样式时无需修改HTML结构
  • 提高页面加载速度 - CSS文件可被浏览器缓存
  • 提高可访问性 - 内容与样式分离，便于屏幕阅读器等辅助技术理解页面结构
  • 提高开发效率 - 一套样式可用于多个页面

练习：结构与表现分离

尝试将以下HTML中的内联样式提取到外部CSS文件中：

<div style="color: red; font-size: 16px; background-color: yellow;"> 这是一个带有内联样式的div元素 </div>

步骤：

  1. 创建一个CSS文件（如style.css）
  2. 在CSS文件中为div元素定义样式
  3. 在HTML文件中使用link标签引入CSS文件
  4. 删除HTML中的内联样式

CSS样式规则

CSS规则集由选择器和声明块组成：

选择器 { 属性1: 属性值1; 属性2: 属性值2; 属性3: 属性值3; }

CSS样式规则的特点：

  • 选择器指定要应用样式的HTML元素
  • 声明块包含一个或多个声明，每个声明由属性和值组成
  • 属性和值之间用冒号分隔
  • 每个声明以分号结束
  • 声明块用花括号包围

练习：编写CSS规则

为以下HTML元素编写CSS规则：

<h1 class="title">网页标题</h1> <p id="intro">这是一个段落</p>

要求：

  • 为class为"title"的h1元素设置颜色为蓝色，字体大小为24px
  • 为id为"intro"的p元素设置字体为微软雅黑，行高为1.5

CSS样式表的引入

CSS样式表可以通过四种方式引入到HTML文档中：

行内式（内联样式）

通过标签的style属性来设置元素的样式：

<标签名 style="属性1:属性值1; 属性2:属性值2; 属性3:属性值3;"> 内容 </标签名>

内嵌式

将CSS代码集中写在HTML文档的head头部标签中，用style标签定义：

<head> <style type="text/css"> 选择器 {属性1:属性值1; 属性2:属性值2; 属性3:属性值3;} </style> </head>

外链式

将所有的样式放在一个或多个以.css为扩展名的外部样式表文件中，通过link标签链接到HTML文档：

<head> <link href="CSS文件的路径" type="text/css" rel="stylesheet" /> </head>

导入式

在style标签内的开头处使用@import语句导入外部样式表文件：

<style type="text/css"> @import url(css文件路径); 或 @import "css文件路径"; /* 在此还可以存放其他CSS样式 */ </style>

注意：实际开发中，推荐使用外链式引入CSS，这样可以实现结构与表现的完全分离，提高代码的可维护性。

练习：使用不同方式引入CSS

创建一个HTML文件，并尝试使用以下方式引入CSS：

  1. 使用行内式为一个div元素设置背景颜色和边框
  2. 使用内嵌式为所有p元素设置字体大小和行高
  3. 创建一个外部CSS文件，使用外链式引入
  4. 在外部CSS文件中，使用导入式引入另一个CSS文件

CSS基础选择器

选择器用于指定要应用样式的HTML元素。

标签选择器

用HTML标签名称作为选择器，为页面中某一类标签指定统一的CSS样式：

标签名 {属性1:属性值1; 属性2:属性值2; 属性3:属性值3; }

类选择器

使用"."（英文点号）进行标识，后面紧跟类名：

.类名 {属性1:属性值1; 属性2:属性值2; 属性3:属性值3; }

ID选择器

使用"#"进行标识，后面紧跟id名：

#id名 {属性1:属性值1; 属性2:属性值2; 属性3:属性值3; }

通配符选择器

用"*"号表示，能匹配页面中所有的元素：

* {属性1:属性值1; 属性2:属性值2; 属性3:属性值3; }

练习：使用不同选择器

为以下HTML结构编写CSS样式：

<div id="header"> <h1 class="title">网站标题</h1> <p class="intro">网站简介</p> </div> <div class="content"> <h2>内容标题</h2> <p>段落内容</p> <ul> <li>列表项1</li> <li>列表项2</li> </ul> </div>

要求：

  • 使用标签选择器为所有p元素设置字体大小
  • 使用类选择器为class为"title"的元素设置颜色
  • 使用ID选择器为id为"header"的元素设置背景颜色
  • 使用通配符选择器设置所有元素的外边距和内边距为0

CSS字体样式属性

CSS字体样式属性用于控制文本的字体显示方式。

font-size属性

用于设置字号，可以使用相对长度单位或绝对长度单位：

p { font-size: 16px; /* 像素单位 */ } h1 { font-size: 2em; /* 相对单位 */ }

font-family属性

用于设置字体，可以指定多个字体作为备选：

body { font-family: "微软雅黑", "宋体", sans-serif; }

font-variant属性

用于设置变体（字体变化），一般用于定义小型大写字母：

p { font-variant: small-caps; /* 小型大写字母 */ }

font-style属性

用于定义字体风格，如设置斜体、倾斜或正常字体：

.italic { font-style: italic; /* 斜体 */ } .oblique { font-style: oblique; /* 倾斜 */ }

font-weight属性

用于设置字体的粗细：

.bold { font-weight: bold; /* 粗体 */ } .light { font-weight: lighter; /* 细体 */ }

font属性（简写）

使用font属性可以一次性设置多个字体属性：

选择器 {font: font-style font-variant font-weight font-size/line-height font-family;}

@font-face属性

CSS3新增属性，用于定义服务器字体：

@font-face { font-family: 字体名称; src: 字体路径; }

CSS文本外观属性

CSS文本外观属性用于控制文本的显示效果。

color属性

用于定义文本的颜色，取值方式：

  • 预定义的颜色值：red, green, blue等
  • 十六进制：#FF0000, #FF6600, #29D794等
  • RGB代码：rgb(255,0,0)或rgb(100%,0%,0%)
h1 { color: #FF0000; /* 红色 */ } p { color: rgb(0, 128, 0); /* 绿色 */ }

多学一招：十六进制颜色值是由#开头的6位十六进制数值组成，每2位为一个颜色分量。当3个分量的2位十六进制数都相同时，可使用CSS缩写。例如，#FF6600可缩写为#F60，#FF0000可缩写为#F00，#FFFFFF可缩写为#FFF。

letter-spacing属性

用于定义字间距，即字符与字符之间的空白：

h1 { letter-spacing: 2px; /* 字间距2像素 */ }

word-spacing属性

用于定义英文单词之间的间距，对中文字符无效：

p { word-spacing: 5px; /* 单词间距5像素 */ }

line-height属性

用于设置行间距（行高）：

p { line-height: 1.5; /* 行高为字体大小的1.5倍 */ }

text-transform属性

用于控制英文字符的大小写：

.uppercase { text-transform: uppercase; /* 全部大写 */ } .capitalize { text-transform: capitalize; /* 首字母大写 */ }

text-decoration属性

用于设置文本的下画线、上画线、删除线等装饰效果：

.underline { text-decoration: underline; /* 下划线 */ } .line-through { text-decoration: line-through; /* 删除线 */ }

text-align属性

用于设置文本内容水平对齐：

.center { text-align: center; /* 居中对齐 */ } .right { text-align: right; /* 右对齐 */ }

注意：text-align属性仅适用于块级元素，对行内元素无效。如果需要对图像设置水平对齐，可以为图像添加一个父标签如p或div，然后对父标签应用text-align属性。

text-indent属性

用于设置首行文本的缩进：

p { text-indent: 2em; /* 首行缩进2个字符宽度 */ }

注意：text-indent属性仅适用于块级元素，对行内元素无效。

white-space属性

可设置空白符的处理方式：

.pre { white-space: pre; /* 预格式化，保留空格空行原样显示 */ } .nowrap { white-space: nowrap; /* 强制文本不能换行 */ }

text-shadow属性

可以为页面中的文本添加阴影效果：

h1 { text-shadow: 2px 2px 4px rgba(0,0,0,0.5); }

参数说明：

  • h-shadow：水平阴影的距离
  • v-shadow：垂直阴影的距离
  • blur：模糊半径
  • color：阴影颜色

多学一招：设置阴影叠加的方法非常简单，只需设置多组阴影参数，中间用逗号隔开即可。

p { font-size: 32px; text-shadow: 10px 10px 10px red, 20px 20px 20px green; /* 红色和绿色的投影叠加 */ }

CSS复合选择器

复合选择器是由两个或多个基础选择器通过不同方式组合而成的。

标签指定式选择器（交集选择器）

由两个选择器构成，第一个为标签选择器，第二个为class选择器或id选择器，两个选择器之间不能有空格：

h3.special { color: red; } p#one { font-weight: bold; }

后代选择器

用来选择元素或元素组的后代，外层标签写在前，内层标签写在后，中间用空格分隔：

div p { color: blue; }

并集选择器

各个选择器通过逗号连接而成，任何形式的选择器都可以作为并集选择器的一部分：

h1, h2, h3 { font-family: "微软雅黑"; }

练习：使用复合选择器

为以下HTML结构编写CSS样式：

<div class="container"> <h1 class="title">主标题</h1> <div class="content"> <h2>副标题</h2> <p>段落内容</p> <ul> <li>列表项1</li> <li class="special">特殊列表项</li> </ul> </div> </div>

要求：

  • 使用后代选择器为content内的p元素设置样式
  • 使用交集选择器为class为special的li元素设置特殊样式
  • 使用并集选择器为h1和h2设置相同的字体

CSS层叠性与继承性

层叠性

层叠性是指多种CSS样式的叠加。当同一个元素被多个选择器选中时，会根据选择器的权重和样式定义的顺序来决定最终应用的样式。

继承性

继承性是指子标签会继承父标签的某些样式，如文本颜色和字号。恰当地使用继承可以简化代码，降低CSS样式的复杂性。

注意：并不是所有的CSS属性都可以继承。例如，下面的属性就不具有继承性：

  • 边框属性
  • 内/外边距属性
  • 背景属性
  • 定位属性
  • 布局属性
  • 元素宽高属性

CSS优先级

当同一个元素被多个选择器选中时，会根据选择器的权重来决定最终应用的样式。

选择器权重

CSS优先级按照以下规则计算：

  • 行内样式：1000
  • ID选择器：100
  • 类选择器、属性选择器、伪类：10
  • 元素选择器、伪元素：1
  • 通用选择器、子选择器、相邻兄弟选择器：0

优先级规则

  • 优先级高的样式会覆盖优先级低的样式
  • 当优先级相同时，后定义的样式会覆盖先定义的样式
  • 使用!important声明的样式具有最高优先级
/* 优先级：1 */ p { color: black; } /* 优先级：10 */ .text { color: blue; } /* 优先级：1+10=11 */ p.text { color: green; } /* 优先级：100 */ #content { color: red; } /* 优先级最高 */ p { color: yellow !important; }

练习：计算选择器优先级

计算以下选择器的优先级权重：

  1. div p
  2. .container .item
  3. #header h1.title
  4. ul li:first-child
  5. div > p

然后，为以下HTML元素编写样式，确保指定的样式能够生效：

<div id="main" class="container"> <h1 class="title">标题</h1> <p class="content">内容</p> </div>

盒子模型

盒子模型是网页布局的基础，只有掌握了盒子模型的各种规律，才能更好地控制网页中各个元素。

认识盒子模型

在CSS中，所有的HTML元素都可以看作盒子。盒子模型包含内容(content)、内边距(padding)、边框(border)和外边距(margin)。

盒子模型组成部分：

  • 内容(content) - 盒子的实际内容，显示文本和图像
  • 内边距(padding) - 清除内容周围的区域，内边距是透明的
  • 边框(border) - 围绕在内边距和内容外的边框
  • 外边距(margin) - 清除边框外的区域，外边距是透明的

注意：虽然盒子模型拥有内边距、边框、外边距、宽度和高度这些基本属性，但是并不要求每个盒子模型都必须定义这些属性。

练习：理解盒子模型

  • 设置内容区域为200px × 100px
  • 添加20px的内边距
  • 添加5px的红色边框
  • 添加30px的外边距

盒子模型基础属性

盒子模型的基础属性包括边框属性、内边距属性、外边距属性、背景属性、宽度属性和高度属性。

边框属性

边框属性用于设置元素边框的样式、宽度和颜色。

/* 边框样式 */ border-style: solid; /* 实线 */ border-style: dashed; /* 虚线 */ border-style: dotted; /* 点线 */ border-style: double; /* 双线 */ /* 边框宽度 */ border-width: 2px; border-width: thin; /* 细边框 */ border-width: medium; /* 中等边框 */ border-width: thick; /* 粗边框 */ /* 边框颜色 */ border-color: #ff0000; border-color: rgb(255, 0, 0); border-color: red; /* 简写属性 */ border: 2px solid #ff0000;

多学一招：为元素添加边框时，如果需要将已有的边框设置为不可见，可使用"border-color:transparent;"样式，此时边框显示透明色。当我们需要边框可见时，再设置相应的颜色即可。通过这种设置方式可以保证元素的区域不发生变化。边框透明样式和取消边框样式不同，取消边框样式时，虽然边框也不可见，但是边框的宽度为0。

注意：使用RGB模式设置颜色时，如果括号里面的数值为百分比，必须把"0"也加上百分号，写作"0%"。

内边距属性

内边距属性用于设置元素内容与边框之间的距离。

/* 单独设置各边内边距 */ padding-top: 10px; padding-right: 15px; padding-bottom: 10px; padding-left: 15px; /* 简写属性 */ padding: 10px; /* 上下左右均为10px */ padding: 10px 15px; /* 上下10px，左右15px */ padding: 10px 15px 20px; /* 上10px，左右15px，下20px */ padding: 10px 15px 20px 25px; /* 上10px，右15px，下20px，左25px */

注意：我们设置的内边距为百分比，都是相对于父元素宽度width的百分比，随父元素width的变化而变化，和高度height无关。

外边距属性

外边距属性用于设置元素与其他元素之间的距离。

/* 单独设置各边外边距 */ margin-top: 10px; margin-right: 15px; margin-bottom: 10px; margin-left: 15px; /* 简写属性 */ margin: 10px; /* 上下左右均为10px */ margin: 10px 15px; /* 上下10px，左右15px */ margin: 10px 15px 20px; /* 上10px，左右15px，下20px */ margin: 10px 15px 20px 25px; /* 上10px，右15px，下20px，左25px */ /* 自动外边距 - 常用于水平居中 */ margin: 0 auto;
这个元素有20px的上下外边距
这个元素使用自动外边距水平居中

背景属性

背景属性用于设置元素的背景效果。

/* 背景颜色 */ background-color: #f0f0f0; /* 背景图像 */ background-image: url('image.jpg'); /* 背景重复 */ background-repeat: no-repeat; /* 不重复 */ background-repeat: repeat-x; /* 水平重复 */ background-repeat: repeat-y; /* 垂直重复 */ background-repeat: repeat; /* 水平和垂直重复（默认） */ /* 背景位置 */ background-position: center top; background-position: 10px 20px; /* 背景固定 */ background-attachment: scroll; /* 图像随页面一起滚动（默认值） */ background-attachment: fixed; /* 图像固定在屏幕上，不随页面滚动 */ /* 简写属性 */ background: #f0f0f0 url('image.jpg') no-repeat center top;
这个元素有背景颜色
这个元素有渐变背景

宽度和高度属性

宽度和高度属性用于设置元素的尺寸。

/* 宽度和高度 */ width: 300px; height: 200px; /* 最大/最小宽度和高度 */ max-width: 100%; min-height: 100px; /* 盒子尺寸计算方式 */ box-sizing: content-box; /* 默认值，宽度和高度只包括内容 */ box-sizing: border-box; /* 宽度和高度包括内容、内边距和边框 */

这个元素有相对宽度和固定高度

计算原则：

  • 盒子的总宽度 = width + 左右内边距之和 + 左右边框宽度之和 + 左右外边距之和
  • 盒子的总高度 = height + 上下内边距之和 + 上下边框宽度之和 + 上下外边距之和

注意：宽度属性width和高度属性height仅适用于块级元素，对行内元素无效。特殊的行内元素<img />和<input />除外。

元素的定位

CSS定位属性可以对元素的位置进行精确控制，实现复杂的布局效果。

定位属性

/* 定位方式 */ position: static; /* 静态定位（默认） */ position: relative; /* 相对定位 */ position: absolute; /* 绝对定位 */ position: fixed; /* 固定定位 */ position: sticky; /* 粘性定位 */ /* 边偏移属性 */ top: 10px; right: 20px; bottom: 30px; left: 40px; /* 堆叠顺序 */ z-index: 10;

静态定位

静态定位是元素的默认定位方式，元素按照正常的文档流进行排列。

静态定位

任何元素在默认状态下都会以静态定位来确定自己的位置。所以当元素没有设置position属性时，并不是说明该元素没有自己的位置，它会遵循默认值显示为静态位置。在静态定位状态下，我们无法通过边偏移属性来改变元素的位置。

相对定位

相对定位的元素相对于其正常位置进行定位，不会脱离文档流。

正常位置
相对定位

绝对定位

绝对定位的元素相对于最近的已定位祖先元素进行定位，如果没有已定位的祖先元素，则相对于初始包含块（通常是body元素）。

  2. 定义多个边偏移属性时，如果同时设置left和right参数值，以left参数值为准；如果同时设置top和bottom参数值，以top参数值为准。
