盒子模型与布局知识手册

盒子模型是网页布局的基础，只有掌握了盒子模型的各种规律，才能更好地控制网页中各个元素。

认识盒子模型

在CSS中，每个HTML元素都可以看作是一个矩形的盒子，这个盒子包含内容(content)、内边距(padding)、边框(border)和外边距(margin)。

盒子模型的构成

  • 内容(content) - 盒子的实际内容，显示文本和图像
  • 内边距(padding) - 内容与边框之间的空白区域
  • 边框(border) - 围绕内容和内边距的边界
  • 外边距(margin) - 盒子与其他元素之间的空白区域

盒子模型可视化演示

边框(border)
内边距(padding)
内容(content)

注意：虽然盒子模型拥有内边距、边框、外边距、宽度和高度这些基本属性，但是并不要求每个盒子模型都必须定义这些属性。

边框属性

边框属性用于设置元素边框的样式、宽度和颜色。

边框样式 (border-style)

定义边框的样式，常用值包括：

  • none - 无边框
  • solid - 实线边框
  • dashed - 虚线边框
  • dotted - 点线边框
  • double - 双线边框

虚线边框示例

边框宽度 (border-width)

定义边框的宽度，可以使用像素(px)、点(pt)、厘米(cm)等单位。

边框颜色 (border-color)

定义边框的颜色，可以使用颜色名称、十六进制值或RGB值。

/* 单独设置各边边框 */ border-top: 2px solid red; border-right: 1px dashed blue; border-bottom: 3px dotted green; border-left: 1px double purple; /* 简写形式 */ border: 2px solid #000;

多学一招：将已有的边框设置为不可见时，可使用border-color: transparent;样式，此时边框显示透明色。通过这种设置方式可以保证元素的区域不发生变化。边框透明样式和取消边框样式不同，取消边框样式时，虽然边框也不可见，但是边框的宽度为0。

注意：使用RGB模式设置颜色时，如果括号里面的数值为百分比，必须把"0"也加上百分号，写作"0%"。

内边距属性

内边距属性用于设置元素内容与边框之间的空白区域。

内边距的设置方式

  • padding: 10px; - 四个方向内边距均为10px
  • padding: 10px 20px; - 上下内边距10px，左右内边距20px
  • padding: 10px 20px 15px; - 上内边距10px，左右内边距20px，下内边距15px
  • padding: 10px 20px 15px 5px; - 上10px，右20px，下15px，左5px
/* 单独设置各边内边距 */ padding-top: 10px; padding-right: 15px; padding-bottom: 20px; padding-left: 5px; /* 简写形式 */ padding: 10px 15px 20px 5px;

注意：我们设置的内边距为百分比，都是相对于父元素宽度width的百分比，随父元素width的变化而变化，和高度height无关。

宽度和高度属性

宽度和高度属性用于控制盒子的大小。

宽度属性 (width)

设置元素内容区域的宽度。

高度属性 (height)

设置元素内容区域的高度。

/* 设置固定尺寸 */ width: 300px; height: 200px; /* 设置相对尺寸 */ width: 50%; height: 80%; /* 设置最大最小尺寸 */ min-width: 200px; max-width: 800px; min-height: 100px; max-height: 500px;

盒子尺寸计算

盒子的总宽度 = width + 左右内边距之和 + 左右边框宽度之和 + 左右外边距之和

盒子的总高度 = height + 上下内边距之和 + 上下边框宽度之和 + 上下外边距之和

注意：宽度属性width和高度属性height仅适用于块级元素，对行内元素无效。特殊的行内元素<img />和<input />除外。

透明属性

透明属性用于设置元素的透明度效果。

rgba模式

rgba模式是在红、绿、蓝三原色的基础上添加了透明度参数，它是rgb模式的延伸。

/* 语法格式 */ rgba(red, green, blue, alpha) /* 示例 */ background-color: rgba(255, 0, 0, 0.5); /* 半透明红色背景 */ color: rgba(0, 0, 0, 0.8); /* 80%不透明黑色文字 */

opacity属性能够使任何元素呈现出透明效果，作用范围要比rgba模式大得多。

/* 语法格式 */ opacity: value; /* 示例 */ opacity: 0.5; /* 50%透明度 */ opacity: 0; /* 完全透明 */ opacity: 1; /* 完全不透明 */

元素的类型

在HTML中，元素主要分为块级元素和行内元素。

块级元素 (Block-level Elements)

  • 特点：每个块元素都会独自占据一行或多行。块元素可以设置宽度、高度、对齐等属性。
  • 作用：常用于网页布局和网页结构的搭建。
  • 常见块元素：<h1>~<h6>、<p>、<div>、<ul>、<ol>、<li>等。

行内元素 (Inline Elements)

  • 特点：不会独自占据一行。不可以设置宽度、高度、对齐等属性。
  • 作用：用于控制页面中文本的样式。
  • 常见行内元素：<strong>、<b>、<em>、<i>、<del>、<s>、<ins>、<u>、<a>、<span>等。

注意：在行内元素中有几个特殊的标签。例如，<img />标签和<input />标签。这些特殊的行内元素可以设置宽度、高度和对齐属性，因此它们也被称为行内块元素。

垂直外边距合并

垂直外边距合并是指当两个垂直相邻的块级元素相遇时，它们的外边距会合并为一个外边距。

相邻块元素垂直外边距合并

当两个相邻的块级元素垂直排列时，它们之间的垂直外边距会合并为单个外边距，其大小为两个外边距中的较大者。

嵌套块元素垂直外边距合并

当一个块级元素包含在另一个块级元素中时，它们的上外边距和/或下外边距也会发生合并。

清除浮动的方法

1. 使用空标签清除浮动

在浮动元素之后添加空标签，并对该空标签应用clear:both;样式。

2. 使用overflow属性清除浮动

为浮动元素的父元素添加overflow:hidden;或overflow:auto;。

overflow属性

overflow属性用于设置当元素的内容溢出其块级容器时如何处理。

overflow属性值

  • visible - 默认值。内容不会被修剪，会呈现在元素框之外
  • hidden - 内容会被修剪，并且其余内容不可见
  • scroll - 内容会被修剪，浏览器会显示滚动条以便查看其余内容

静态定位

静态定位是元素的默认定位方式。

静态定位的特点

  • 任何元素在默认状态下都会以静态定位来确定自己的位置
  • 当元素没有设置position属性时，并不是说明该元素没有自己的位置，它会遵循默认值显示为静态位置
  • 在静态定位状态下，无法通过边偏移属性来改变元素的位置
/* 默认值，通常不需要显式设置 */ position: static;

相对定位

相对定位是相对于元素在正常文档流中的原始位置进行定位。

相对定位的特点

  • 元素会相对于其正常位置进行偏移
  • 元素原本所占的空间会保留
  • 不会影响其他元素的位置
.relative-box { position: relative; top: 20px; left: 30px; }

绝对定位

绝对定位是相对于最近的已定位祖先元素进行定位，如果没有已定位的祖先元素，则相对于初始包含块（通常是html元素）进行定位。

绝对定位的特点

  • 元素会脱离正常的文档流
  • 元素原本所占的空间不会保留
  • 位置相对于最近的已定位祖先元素
  • 如果没有已定位的祖先元素，则相对于初始包含块
.container { position: relative; /* 创建定位上下文 */ } .absolute-box { position: absolute; top: 10px; right: 20px; }

z-index属性

z-index属性用于控制定位元素的堆叠顺序。

z-index属性的特点

  • 拥有更高z-index值的元素会覆盖较低值的元素
  • 仅对定位元素（position值不是static的元素）生效
  • 可以是正数、负数或0
  • 默认值为auto，相当于0
.box1 { position: absolute; z-index: 1; } .box2 { position: absolute; z-index: 2; /* 这个元素会覆盖在box1上面 */ }

注意：z-index属性仅对定位元素生效。

响应式设计

响应式设计使网页能够适应不同设备和屏幕尺寸。

媒体查询

媒体查询是响应式设计的核心技术，允许根据设备特性应用不同的CSS规则。

/* 针对小屏幕设备的样式 */ @media (max-width: 768px) { .container { flex-direction: column; } .sidebar { width: 100%; } } /* 针对打印的样式 */ @media print { .sidebar { display: none; } body { font-size: 12pt; } }

常用断点

设备类型 屏幕宽度 媒体查询
手机 < 768px @media (max-width: 767px)
平板 768px - 1024px @media (min-width: 768px) and (max-width: 1024px)
桌面 > 1024px @media (min-width: 1025px)

响应式单位

单位 说明 示例
% 相对于父元素的百分比 width: 50%;
vw / vh 视口宽度/高度的百分比 width: 50vw; (视口宽度的50%)
rem 相对于根元素(html)的字体大小 font-size: 1.5rem;
em 相对于当前元素的字体大小 margin: 2em;
