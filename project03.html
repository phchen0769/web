CSS知识点完整参考手册

CSS（层叠样式表）是一种用于描述HTML文档呈现方式的样式表语言。它控制网页的布局、颜色、字体等视觉表现。

CSS的作用

  • 内容与表现分离 - HTML负责结构，CSS负责样式
  • 提高开发效率 - 一套样式可用于多个页面
  • 增强可维护性 - 修改样式无需改动HTML结构
  • 实现响应式设计 - 使网页适应不同设备屏幕

CSS的版本

  • CSS1 - 1996年发布，包含基础样式属性
  • CSS2 - 1998年发布，增加定位、媒体类型等
  • CSS2.1 - 2011年发布，修正CSS2的问题
  • CSS3 - 模块化发布，包含众多新特性

结构与表现分离

结构与表现分离是Web标准的核心思想之一，指的是将HTML文档的结构与CSS样式分开处理。

知识点详解

  • 结构：指网页的内容与语义，由HTML标签负责。例如，使用<h1>定义标题、<p>定义段落、<ul>定义列表。结构是网页的骨架。
  • 表现：指网页的外观样式，由CSS负责。例如，设置颜色、字体、布局等。表现是网页的外衣。

为什么要将结构和表现分离？

  • 提高代码可维护性 - 修改样式时无需修改HTML结构
  • 提高页面加载速度 - CSS文件可被浏览器缓存
  • 提高可访问性 - 内容与样式分离，便于屏幕阅读器等辅助技术理解页面结构
  • 提高开发效率 - 一套样式可用于多个页面

CSS样式规则

CSS规则集由选择器和声明块组成：

选择器 { 属性1: 属性值1; 属性2: 属性值2; 属性3: 属性值3; }

知识点详解

  • 选择器：用于指定要应用样式的HTML元素。
  • 声明块：由花括号{}包围，包含一条或多条声明。
  • 声明：由属性和值组成，格式为属性: 值;。每条声明以分号结束。

示例：p { color: blue; font-size: 14px; }

  • p 是选择器
  • { color: blue; font-size: 14px; } 是声明块
  • color: blue; 是一条声明，其中color是属性，blue是值

CSS样式表的引入

CSS样式表可以通过四种方式引入到HTML文档中：

方式 语法 说明 应用场景
行内式 <标签名 style="属性:值;"> 直接写在HTML标签的style属性中 优先级最高，但不利于维护，不推荐大量使用
内嵌式 <style>选择器 {属性:值;}</style> 写在HTML的head标签内的style标签中 适用于单页面、样式较少的情况
外链式 <link href="style.css" rel="stylesheet"> 在单独的.css文件中写样式，通过link标签引入 最佳实践，利于维护和复用，最适合中大型项目
导入式 @import url("style.css"); 在style标签内使用@import指令引入 有性能和使用限制，不如外链式常用

注意：实际开发中，推荐使用外链式引入CSS，这样可以实现结构与表现的完全分离，提高代码的可维护性。

CSS基础选择器

选择器用于指定要应用样式的HTML元素。

标签选择器

用HTML标签名称作为选择器，为页面中某一类标签指定统一的CSS样式：

标签名 {属性1:属性值1; 属性2:属性值2; 属性3:属性值3; }

类选择器

使用"."（英文点号）进行标识，后面紧跟类名：

.类名 {属性1:属性值1; 属性2:属性值2; 属性3:属性值3; }

特点：一个元素可以有多个类，一个类可以被多个元素使用。

ID选择器

使用"#"进行标识，后面紧跟id名：

#id名 {属性1:属性值1; 属性2:属性值2; 属性3:属性值3; }

特点：同一个ID在同一页面中只能使用一次。

通配符选择器

用"*"号表示，能匹配页面中所有的元素：

* {属性1:属性值1; 属性2:属性值2; 属性3:属性值3; }

应用：常用于重置浏览器的默认样式。

CSS字体样式属性

CSS字体样式属性用于控制文本的字体显示方式。

font-size属性

用于设置字号，可以使用相对长度单位或绝对长度单位：

p { font-size: 16px; /* 像素单位 */ } h1 { font-size: 2em; /* 相对单位 */ }

font-family属性

用于设置字体，可以指定多个字体作为备选：

body { font-family: "微软雅黑", "宋体", sans-serif; }

注意：字体名含空格需用引号包围。

font-weight属性

用于设置字体的粗细：

.bold { font-weight: bold; /* 粗体 */ } .light { font-weight: lighter; /* 细体 */ }

font-style属性

用于定义字体风格，如设置斜体、倾斜或正常字体：

.italic { font-style: italic; /* 斜体 */ } .oblique { font-style: oblique; /* 倾斜 */ }

font-variant属性

用于设置变体（字体变化），一般用于定义小型大写字母：

p { font-variant: small-caps; /* 小型大写字母 */ }

font属性（简写）

使用font属性可以一次性设置多个字体属性：

选择器 {font: font-style font-variant font-weight font-size/line-height font-family;}

注意：顺序必须严格按照上述格式。

@font-face属性

CSS3新增属性，用于定义服务器字体：

@font-face { font-family: 字体名称; src: 字体路径; }

CSS文本外观属性

CSS文本外观属性用于控制文本的显示效果。

color属性

用于定义文本的颜色，取值方式：

  • 预定义的颜色值：red, green, blue等
  • 十六进制：#FF0000, #FF6600, #29D794等
  • RGB代码：rgb(255,0,0)或rgb(100%,0%,0%)
h1 { color: #FF0000; /* 红色 */ } p { color: rgb(0, 128, 0); /* 绿色 */ }

多学一招：十六进制颜色值是由#开头的6位十六进制数值组成，每2位为一个颜色分量。当3个分量的2位十六进制数都相同时，可使用CSS缩写。例如，#FF6600可缩写为#F60，#FF0000可缩写为#F00，#FFFFFF可缩写为#FFF。

text-align属性

用于设置文本内容水平对齐：

.center { text-align: center; /* 居中对齐 */ } .right { text-align: right; /* 右对齐 */ }

注意：text-align属性仅适用于块级元素，对行内元素无效。如果需要对图像设置水平对齐，可以为图像添加一个父标签如p或div，然后对父标签应用text-align属性。

text-indent属性

用于设置首行文本的缩进：

p { text-indent: 2em; /* 首行缩进2个字符宽度 */ }

注意：text-indent属性仅适用于块级元素，对行内元素无效。

line-height属性

用于设置行间距（行高）：

p { line-height: 1.5; /* 行高为字体大小的1.5倍 */ }

letter-spacing属性

用于定义字间距，即字符与字符之间的空白：

h1 { letter-spacing: 2px; /* 字间距2像素 */ }

word-spacing属性

用于定义英文单词之间的间距，对中文字符无效：

p { word-spacing: 5px; /* 单词间距5像素 */ }

text-transform属性

用于控制英文字符的大小写：

.uppercase { text-transform: uppercase; /* 全部大写 */ } .capitalize { text-transform: capitalize; /* 首字母大写 */ }

text-decoration属性

用于设置文本的下画线、上画线、删除线等装饰效果：

.underline { text-decoration: underline; /* 下划线 */ } .line-through { text-decoration: line-through; /* 删除线 */ }

white-space属性

可设置空白符的处理方式：

.pre { white-space: pre; /* 预格式化，保留空格空行原样显示 */ } .nowrap { white-space: nowrap; /* 强制文本不能换行 */ }

text-shadow属性

可以为页面中的文本添加阴影效果：

h1 { text-shadow: 2px 2px 4px rgba(0,0,0,0.5); }

参数说明：

  • h-shadow：水平阴影的距离
  • v-shadow：垂直阴影的距离
  • blur：模糊半径
  • color：阴影颜色

多学一招：设置阴影叠加的方法非常简单，只需设置多组阴影参数，中间用逗号隔开即可。

p { font-size: 32px; text-shadow: 10px 10px 10px red, 20px 20px 20px green; /* 红色和绿色的投影叠加 */ }

CSS复合选择器

复合选择器是由两个或多个基础选择器通过不同方式组合而成的。

标签指定式选择器（交集选择器）

由两个选择器构成，第一个为标签选择器，第二个为class选择器或id选择器，两个选择器之间不能有空格：

h3.special { color: red; } p#one { font-weight: bold; }

后代选择器

用来选择元素或元素组的后代，外层标签写在前，内层标签写在后，中间用空格分隔：

div p { color: blue; }

并集选择器

各个选择器通过逗号连接而成，任何形式的选择器都可以作为并集选择器的一部分：

h1, h2, h3 { font-family: "微软雅黑"; }

CSS层叠性与继承性

层叠性

层叠性是指多种CSS样式的叠加。当同一个元素被多个选择器选中时，会根据选择器的权重和样式定义的顺序来决定最终应用的样式。

继承性

继承性是指子标签会继承父标签的某些样式，如文本颜色和字号。恰当地使用继承可以简化代码，降低CSS样式的复杂性。

注意：并不是所有的CSS属性都可以继承。例如，下面的属性就不具有继承性：

  • 边框属性
  • 内/外边距属性
  • 背景属性
  • 定位属性
  • 布局属性
  • 元素宽高属性

CSS优先级

当同一个元素被多个选择器选中时，会根据选择器的权重来决定最终应用的样式。

选择器权重

CSS优先级按照以下规则计算：

  • 行内样式：1000
  • ID选择器：100
  • 类选择器、属性选择器、伪类：10
  • 元素选择器、伪元素：1
  • 通用选择器、子选择器、相邻兄弟选择器：0

优先级规则

  • 优先级高的样式会覆盖优先级低的样式
  • 当优先级相同时，后定义的样式会覆盖先定义的样式
  • 使用!important声明的样式具有最高优先级
/* 优先级：1 */ p { color: black; } /* 优先级：10 */ .text { color: blue; } /* 优先级：1+10=11 */ p.text { color: green; } /* 优先级：100 */ #content { color: red; } /* 优先级最高 */ p { color: yellow !important; }
