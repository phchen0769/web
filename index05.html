项目7："视频8"首页制作

本项目专注于网页中音视频嵌入以及CSS3过渡、变形和动画效果的应用，通过实际案例"视频8"首页的制作，掌握现代网页设计中的动态效果实现。

项目背景

随着互联网的不断发展，以短视频、网络电视剧、网络电影为代表的网络视频创作高速增长，市场需求日益旺盛。为了契合市场机遇，"视频8"负责人致电HC公司王经理，计划制作一个网站。

技术栈： HTML5、CSS3（过渡、变形、动画）、JavaScript

学习目标 / Target

  • 掌握网页中嵌入视频和音频的方法
  • 理解CSS3过渡属性的原理和应用
  • 学会使用CSS3变形效果
  • 掌握CSS3动画的制作方法
  • 能够综合运用所学知识制作动态网页

项目描述 / Summary

接到王经理安排的任务后，小思制定了近期的项目计划：首先了解网站音频、视频的嵌入方式，然后学习CSS3新增的过渡、变形、动画等属性。最后运用所学知识，制作一个带有视频和过渡效果的网站。

本项目只演示"视频8"首页的制作过程，但涵盖的技术可以应用于各种现代网页设计中。

任务7-1：网页中嵌入视频和音频

需求分析

在网络传输速度越来越快的今天，音频和视频嵌入技术已经被越来越广泛的应用在网页设计中，比起静态的图片和文字，音频和视频可以为用户提供更直观、丰富的信息。

学习目标

掌握在网页中嵌入视频和音频的方法，了解浏览器兼容性问题，能够为不同浏览器选择兼容性最好的视频格式。

音频、视频嵌入技术概述

了解使用第三方插件和使用HTML5中的标签嵌入视频和音频的区别，能够说出它们的优缺点。

HTML5音视频嵌入 vs 第三方插件

  • HTML5音视频标签： 原生支持、无需插件、轻量级、标准化
  • 第三方插件（如Flash）： 兼容老浏览器、功能丰富但逐渐被淘汰

练习：比较嵌入方式

列出HTML5音视频标签与第三方插件（如Flash）在以下方面的对比：

  • 兼容性
  • 性能
  • 安全性
  • 移动设备支持
  • 开发复杂度

嵌入视频

掌握嵌入视频的方法，能够在网页中嵌入视频。

<video width="640" height="360" controls> <source src="movie.mp4" type="video/mp4"> <source src="movie.webm" type="video/webm"> <source src="movie.ogg" type="video/ogg"> 您的浏览器不支持HTML5视频标签。 </video>

练习：嵌入视频

创建一个包含以下功能的视频播放器：

  • 支持MP4和WebM格式
  • 显示播放控件
  • 设置合适的宽度和高度
  • 添加备用文本，在不支持HTML5视频的浏览器中显示

嵌入音频

掌握嵌入音频的方法，能够在网页中嵌入音频。

<audio controls> <source src="audio.mp3" type="audio/mpeg"> <source src="audio.ogg" type="audio/ogg"> <source src="audio.wav" type="audio/wav"> 您的浏览器不支持HTML5音频标签。 </audio>

练习：嵌入音频

创建一个音频播放器，要求：

  • 支持MP3和OGG格式
  • 显示播放控件
  • 添加循环播放属性
  • 设置预加载属性为"metadata"

浏览器对音视频文件的兼容性

了解各浏览器对视频和音频文件格式的兼容情况，能够针对不同浏览器，选择兼容性最好的视频格式。

注意：不同浏览器对音视频格式的支持情况不同，通常需要提供多种格式以确保兼容性。

练习：兼容性处理

研究当前主流浏览器（Chrome、Firefox、Safari、Edge）对以下格式的支持情况：

  • MP4/H.264
  • WebM/VP9
  • OGG/Theora
  • MP3
  • WAV
  • OGG Vorbis

根据研究结果，为网站选择最合适的音视频格式组合。

控制视频宽高

掌握控制视频宽高的方法，能够为网页中的视频设置宽高。

<video width="800" height="450" controls> <source src="movie.mp4" type="video/mp4"> </video>

注意：通过width属性和height属性来缩放视频，这样的视频即使在页面上看起来很小，但它的原始大小依然没变，因此在实际工作中要运用视频处理软件（例如"格式工厂"）对视频进行压缩。

练习：视频尺寸控制

创建一个响应式视频播放器：

  • 使用CSS设置视频最大宽度为100%
  • 保持视频的宽高比
  • 在不同屏幕尺寸下都能正常显示

任务7-2：网页中添加过渡效果

需求分析

CSS提供了强大的过渡属性（CSS3属性），它可以在不使用Flash动画或者JavaScript脚本的情况下，为元素从一种样式转变为另一种样式时添加效果。例如，渐显、渐隐、速度的变化等。

学习目标

掌握CSS3过渡属性的使用方法，能够在网页中应用过渡效果。

transition-property 属性

掌握设置transition-property属性的用法，能够在网页中运用。

.demo { transition-property: width, background-color; }
悬停查看效果

练习：transition-property应用

创建一个元素，为其设置过渡效果：

  • 当鼠标悬停时，宽度和高度同时变化
  • 背景颜色和文字颜色也应用过渡效果
  • 使用transition-property指定需要过渡的属性

transition-duration 属性

掌握设置transition-duration属性的用法，并能够在网页中运用。

.demo { transition-property: all; transition-duration: 0.5s; }

练习：过渡时长控制

创建三个相同的元素，为它们设置不同的过渡时长：

  • 第一个元素：过渡时长为0.3秒
  • 第二个元素：过渡时长为1秒
  • 第三个元素：过渡时长为2秒

观察不同过渡时长的视觉效果差异。

transition-timing-function 属性

掌握设置transition-timing-function属性的用法，并能够在网页中运用。

.demo { transition-timing-function: ease-in-out; } /* 其他可选值 */ .demo1 { transition-timing-function: linear; } .demo2 { transition-timing-function: ease-in; } .demo3 { transition-timing-function: ease-out; } .demo4 { transition-timing-function: cubic-bezier(0.68, -0.55, 0.27, 1.55); }

练习：过渡时间函数

创建四个相同的元素，为它们设置不同的过渡时间函数：

  • linear - 匀速运动
  • ease-in - 加速运动
  • ease-out - 减速运动
  • ease-in-out - 先加速后减速

观察不同时间函数对过渡效果的影响。

transition-delay 属性

掌握设置transition-delay属性的用法，并能够在网页中运用。

.demo { transition-delay: 0.5s; }

练习：过渡延迟

创建一个元素，为其设置过渡延迟效果：

  • 当鼠标悬停时，过渡效果延迟0.5秒开始
  • 当鼠标移出时，过渡效果立即开始
  • 尝试设置不同的进入和离开延迟

transition 属性（简写）

掌握设置transition属性用法，并能够在网页中运用。

.demo { /* property duration timing-function delay */ transition: all 0.5s ease-in-out 0.1s; }

注意：如果使用transition简写属性设置多种过渡效果，需要为每个过渡属性集中指定所有的值，并且使用逗号进行分隔。

练习：transition简写应用

使用transition简写属性为一个元素设置多个过渡效果：

  • 宽度变化：0.3秒，ease-in-out
  • 背景颜色变化：0.5秒，linear，延迟0.1秒
  • 边框圆角变化：1秒，ease-out

任务7-3：在网页中添加变形效果

需求分析

在CSS3中，通过变形可以对元素进行平移、缩放、倾斜和旋转等操作。同时变形可以和过渡属性结合，实现一些绚丽网页动画效果。网页中添加的变形效果主要包括2D变形和3D变形两种。

学习目标

掌握2D和3D变形的方法，能够为网页元素设置变形效果，并能够列举各种变形效果。

2D变形

了解2D变形包含哪些变形效果，能够一一列举。掌握2D变形的方法，能够为网页模块设置2D变形效果。

/* 平移 */ transform: translate(50px, 100px); /* 缩放 */ transform: scale(1.5, 1.5); /* 旋转 */ transform: rotate(45deg); /* 倾斜 */ transform: skew(30deg, 20deg); /* 多重变换 */ transform: translate(50px, 100px) rotate(45deg) scale(1.5);
悬停查看2D变形

注意：translate()中参数值的单位不可以省略，否则平移命令将不起作用。如果一个元素需要设置多种变形效果，可以使用空格把多个变形属性值隔开。

练习：2D变形应用

创建一个包含多种2D变形效果的元素：

  • 初始状态：正常显示
  • 鼠标悬停时：向右平移50px，旋转45度，放大1.2倍
  • 使用transition使变形过程平滑

3D变形

了解3D变形包含哪些变形效果，并能够一一列举这些效果。掌握3D变形的方法，能够对网页中的元素应用3D变形效果。

/* 绕X轴旋转 */ transform: rotateX(60deg); /* 绕Y轴旋转 */ transform: rotateY(60deg); /* 绕Z轴旋转 */ transform: rotateZ(60deg); /* 3D平移 */ transform: translate3d(50px, 100px, 50px); /* 透视 */ perspective: 500px;

注意：rotateZ()函数和rotateX()函数、rotateY()函数功能一样，区别在于rotateZ()函数用于指定一个元素围绕Z轴旋转。如果仅从视觉角度上看，rotateZ()函数让元素顺时针或逆时针旋转，与rotate()效果等同，但rotateZ不是在2D平面上的旋转。

练习：3D变形应用

创建一个3D卡片翻转效果：

  • 创建一个包含正面和背面的卡片容器
  • 使用transform-style: preserve-3d启用3D空间
  • 使用rotateY实现卡片翻转效果
  • 添加透视效果使3D效果更明显

任务7-4：网页中添加动画效果

需求分析

过渡和变形只能设置元素的变换过程，并不能对过程中的某一环节进行精确控制。例如，过渡和变形实现的动态效果不能够重复播放。为了实现更加丰富的动画效果，CSS3提供了animation属性，使用animation属性可以定义复杂的动画效果。

学习目标

掌握CSS3动画属性的使用方法，能够定义关键帧动画并在网页中应用。

@keyframes 规则

掌握@keyframes规则的用法，能够配合动画属性为网页添加动画效果。

@keyframes example { 0% { background-color: red; left: 0px; top: 0px; } 25% { background-color: yellow; left: 200px; top: 0px; } 50% { background-color: blue; left: 200px; top: 200px; } 75% { background-color: green; left: 0px; top: 200px; } 100% { background-color: red; left: 0px; top: 0px; } } .demo { animation-name: example; animation-duration: 4s; }

注意：Internet Explorer 9浏览器以及更早的版本，不支持@keyframe规则和animation属性。

练习：创建关键帧动画

创建一个简单的关键帧动画：

  • 定义从0%到100%的关键帧
  • 包含位置变化和颜色变化
  • 应用到一个div元素上
  • 设置动画时长为3秒

animation-name 属性

掌握animation-name属性的用法，能够在网页中应用。

.demo { animation-name: slidein; }

练习：动画命名与应用

创建两个不同的关键帧动画：

  • 第一个动画名为"fadeIn"，实现淡入效果
  • 第二个动画名为"bounce"，实现弹跳效果
  • 创建两个元素，分别应用这两个动画

animation-duration 属性

掌握animation-duration属性的用法，能够在网页中应用。

.demo { animation-duration: 2s; }

练习：动画时长控制

创建三个相同的动画元素：

  • 第一个元素：动画时长为1秒
  • 第二个元素：动画时长为3秒
  • 第三个元素：动画时长为5秒

观察不同动画时长的效果差异。

animation-timing-function 属性

掌握animation-timing-function属性的用法，能够在网页中应用。

.demo { animation-timing-function: ease-in-out; }

练习：动画时间函数

创建四个相同的动画元素，为它们设置不同的时间函数：

  • linear - 匀速
  • ease - 默认，慢-快-慢
  • ease-in - 慢开始
  • ease-out - 慢结束

观察不同时间函数对动画效果的影响。

animation-delay 属性

掌握animation-delay属性的用法，能够在网页中应用。

.demo { animation-delay: 2s; }

练习：动画延迟

创建一个动画序列：

  • 创建三个相同的动画元素
  • 第一个元素：无延迟
  • 第二个元素：延迟1秒
  • 第三个元素：延迟2秒
  • 实现连续播放的效果

animation-iteration-count 属性

掌握animation-iteration-count属性的用法，能够在网页中应用。

.demo { /* 播放动画的次数 */ animation-iteration-count: 3; /* 指定动画循环播放 */ animation-iteration-count: infinite; }
无限循环动画

练习：动画循环控制

创建三个动画元素：

  • 第一个元素：播放1次
  • 第二个元素：播放3次
  • 第三个元素：无限循环

animation-direction 属性

掌握animation-direction属性的用法，能够在网页中应用。

.demo { /* 动画正常播放 */ animation-direction: normal; /* alternate属性值会使动画在奇数次数（1、3、5等）正常播放，而在偶数次数（2、4、6等）逆向播放 */ animation-direction: alternate; }

练习：动画方向控制

创建两个动画元素：

  • 第一个元素：使用normal方向
  • 第二个元素：使用alternate方向
  • 观察两种方向的差异

animation 属性（简写）

掌握animation属性的用法，能够在网页中应用。

.demo { /* name duration timing-function delay iteration-count direction */ animation: example 3s ease-in-out 1s infinite alternate; }

练习：animation简写应用

使用animation简写属性创建一个复杂的动画：

  • 动画名称：complexAnimation
  • 时长：4秒
  • 时间函数：ease-in-out
  • 延迟：0.5秒
  • 循环次数：无限
  • 方向：交替
