列表与超链接知识手册

列表和超链接是网页中必不可少的元素，能够有效地组织和连接网页内容，提高用户体验。

列表与超链接的作用

  • 有序组织内容 - 列表可以将相关内容分组展示，使页面结构清晰
  • 提高可读性 - 列表形式便于用户快速扫描和理解内容
  • 连接相关内容 - 超链接可以将不同页面或内容区域连接起来
  • 增强导航功能 - 超链接是实现网站导航的基础

无序列表

无序列表用于表示一组没有特定顺序的项目列表。

无序列表的基本语法

<ul> <li>列表项一</li> <li>列表项二</li> <li>列表项三</li> </ul>

无序列表的特点

  • 列表项没有特定的顺序
  • 默认使用实心圆点作为项目符号
  • 可以通过CSS自定义项目符号样式

无序列表示例

  • 前端开发课程
  • 后端开发课程
  • UI/UX设计课程
  • 数据科学课程

注意：

  1. 无序列表拥有type属性，用于指定不同的列表项目符号。但是HTML中不赞成使用type属性，一般通过CSS样式属性替代。
  2. 在<ul></ul>中只能嵌套<li></li>，<li>与</li>之间相当于一个容器，可以嵌套其他元素。

有序列表

有序列表用于表示一组有特定顺序或步骤的项目列表。

有序列表的基本语法

<ol> <li>第一步</li> <li>第二步</li> <li>第三步</li> </ol>

有序列表的特点

  • 列表项有特定的顺序
  • 默认使用数字作为项目编号
  • 可以通过CSS自定义编号样式

有序列表示例

  1. 注册优课教育账号
  2. 选择感兴趣的课程
  3. 完成课程学习
  4. 参加在线测试
  5. 获得结业证书

注意：有序列表ol和列表项li拥有一些标签属性。例如，type、start、value等，但在网页中不会使用，一般通过CSS样式属性替代，读者了解即可。

定义列表

定义列表用于表示术语及其定义的列表。

定义列表的基本语法

<dl> <dt>术语一</dt> <dd>术语一的定义或描述</dd> <dt>术语二</dt> <dd>术语二的定义或描述</dd> </dl>

定义列表的特点

  • 由术语(dt)和定义(dd)组成
  • 没有项目符号或编号
  • 常用于词汇表、FAQ等场景

定义列表示例

HTML
超文本标记语言，用于创建网页结构
CSS
层叠样式表，用于控制网页的外观和布局
JavaScript
一种脚本语言，用于实现网页的交互功能

应用技巧：在网页设计中，定义列表常用于实现图文混排效果。<dt>标签中插入图片，<dd>标签中放入对图片解释说明的文字。

CSS控制列表样式 - list-style复合属性

list-style属性是设置列表样式的简写属性，可以同时设置列表的项目符号类型、位置和图像。

list-style属性的语法

/* 语法格式 */ list-style: list-style-type list-style-position list-style-image; /* 示例 */ list-style: square inside; /* 方形符号，内部位置 */ list-style: none; /* 无项目符号 */ list-style: url('marker.png') outside; /* 使用图像作为项目符号，外部位置 */

list-style-type属性

设置列表项标记的类型：

  • disc - 实心圆（默认）
  • circle - 空心圆
  • square - 实心方块
  • decimal - 数字
  • lower-roman - 小写罗马数字
  • upper-roman - 大写罗马数字
  • lower-alpha - 小写字母
  • upper-alpha - 大写字母
  • none - 无标记

list-style-position属性

设置列表项标记的位置：

  • inside - 标记位于列表项内容内部
  • outside - 标记位于列表项内容外部（默认）

list-style属性示例

  • 内部方形标记 - 标记位于内容内部
  • 内部方形标记 - 标记位于内容内部
  • 外部小写字母标记 - 标记位于内容外部
  • 外部小写字母标记 - 标记位于内容外部

背景图像定义列表项目符号

除了使用预定义的项目符号，还可以使用背景图像自定义列表项目符号。

使用背景图像定义项目符号的方法

ul { list-style: none; /* 移除默认项目符号 */ padding-left: 0; /* 移除默认内边距 */ } li { padding-left: 30px; /* 为背景图像留出空间 */ background-image: url('marker.png'); /* 设置背景图像 */ background-repeat: no-repeat; /* 不重复背景图像 */ background-position: left center; /* 背景图像位置 */ background-size: 20px 20px; /* 背景图像尺寸 */ }

背景图像项目符号示例

  • 自定义蓝色圆形标记
  • 自定义绿色方形标记
  • 自定义红色三角形标记

创建超链接

超链接是网页中用于从一个页面跳转到另一个页面的元素。

超链接的基本语法

<a href="目标URL">链接文本</a>

超链接的类型

  • 外部链接 - 链接到其他网站
  • 内部链接 - 链接到同一网站的其他页面
  • 空链接 - 暂时没有确定链接目标
  • 下载链接 - 链接到可下载文件
  • 电子邮件链接 - 打开电子邮件客户端

超链接示例

外部链接 内部链接 空链接 下载链接 电子邮件链接

注意：

  1. 暂时没有确定链接目标时，通常将<a>标签的href属性值定义为"#"(即href="#")，表示该链接暂时为一个空链接。
  2. 在网页中，不仅可以为文本创建超链接，还可以为图像、表格、音频、视频等网页元素添加超链接。

链接伪类控制超链接

CSS提供了链接伪类，可以为超链接的不同状态设置不同的样式。

链接伪类

  • a:link - 未访问的链接
  • a:visited - 已访问的链接
  • a:hover - 鼠标悬停在链接上
  • a:active - 链接被点击时
/* 未访问的链接 */ a:link { color: blue; text-decoration: none; } /* 已访问的链接 */ a:visited { color: purple; } /* 鼠标悬停 */ a:hover { color: red; text-decoration: underline; } /* 链接被点击时 */ a:active { color: green; }

链接伪类示例

未访问链接 已访问链接 悬停状态 点击状态

注意：

  1. 同时使用链接的4种伪类时，需要按照a:link、a:visited、a:hover和a:active的顺序书写，否则定义的样式可能不起作用。
  2. 除了文本样式之外，链接伪类还可用于控制超链接的背景、边框等样式。
